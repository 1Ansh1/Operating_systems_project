SINGLE SIMULATION :

    A - Using a workload file (like simple_trace.txt):

        # Run FIFO with 3 frames on your simple trace
        python main.py --workload_file workloads/simple_trace.txt --single FIFO 3

        # Run LRU with 4 frames on the same trace
        python main.py --workload_file workloads/simple_trace.txt --single LRU 4

        # Run Optimal with 3 frames
        python main.py --workload_file workloads/simple_trace.txt --single Optimal 3

        # Run MGLRU with 4 frames
        python main.py --workload_file workloads/simple_trace.txt --single MGLRU 4

    B - Using a freshly generated workload:

        # Run LRU with 10 frames on a 100-request random workload
        python main.py --generate_workload random 100 20 --single LRU 10

        # Run MGLRU with 20 frames on a 500-request locality workload
        python main.py --generate_workload locality 500 50 --single MGLRU 20

FULL COMPARISION:
    A - Using a workload file:

        # Run all algorithms for 1 through 15 frames on your simple trace
        python main.py --workload_file workloads/simple_trace.txt --compare 1 15

    B - Using a freshly generated workload:

        # It tests all algorithms from 1 to 30 frames.
        python main.py --generate_workload locality 500 50 --compare 1 30

        # This is the 10,000-request "stress test".
        python main.py --generate_workload locality 10000 50 --compare 1 30

        # A quick test with a "sequential" workload.
        python main.py --generate_workload sequential 1000 20 --compare 5 25
